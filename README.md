🛡️ PortScan v2.0 - Advanced Network Scanner
PortScan v2.0, ağ üzerindeki hedef sistemlerin açık portlarını tespit etmek, servis analizi yapmak ve bulguları görsel bir HTML raporuna dönüştürmek için tasarlanmış, Python tabanlı ve çok iş parçacıklı (multithreaded) bir ağ tarama aracıdır.

✨ Özellikler
Yüksek Performans: Python threading ve queue modülleri ile eşzamanlı tarama.

Esnek Hedef Tanımlama: Tekil IP veya IP aralığı (örn: 192.168.1.1-10) desteği.

Protokol Desteği: Hem TCP (Tam Bağlantı) hem de UDP tarama seçenekleri.

Banner Grabbing: Açık portlardan servis bilgilerini ve versiyon bannerlarını yakalama.

Görsel Raporlama: Modern, karanlık tema destekli ve interaktif HTML rapor çıktısı.

Renkli CLI: Kullanıcı dostu, renk kodlu terminal arayüzü.

🚀 Kurulum
Herhangi bir harici kütüphane gerektirmez. Sadece Python 3.x yüklü olması yeterlidir.
